Your user token

We now need your user token in order to read messages of your channels. The token will only be used to read messages and is destroyed immediately once the application is shut down.

If you need help to retrieve your user token, here is how to do it :

  1. Open https://discordapp.com/channels/@me and be sure you're logged in
  2. While on the Discord page (not this one), open the Developer Tools tab using Ctrl + Shift + I
  3. In the Developer Tools, open the "Application" Tab. Then under the "Storage" section, open the "Local Storage" drawer by clicking on the small arrow
  4. Click on the "https://discordapp.com" line that appeared below. This should a list of Key - Value pairs.
  5. Find the line with the "token" Key. Select its Value and copy it. (Should be a 60~62 characters string).
  6. Paste it in the above field, and then click Submit.
